# Titanic Survival Prediction using Neural Networks

## Overview
This project uses neural network models built with TensorFlow and Keras to predict passenger survival on the Titanic dataset.

## Features
- Data preprocessing
- Feature engineering
- Neural network model development
- Hyperparameter tuning
- Model evaluation
- Data visualization

## Technologies
- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Titanic dataset from Kaggle.
